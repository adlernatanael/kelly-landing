# Kelly — Landing Page

## Como usar

1. **Logo**: copie `assets/logo-k.png` do app para `screenshots/logo-k.png`
2. **Screenshots**: tire prints do app e salve como `screenshots/1.png`, `screenshots/2.png` ... `screenshots/6.png`
3. **APK**: quando gerar o build, coloque o arquivo como `kelly.apk` na raiz
4. **Deploy**: suba para GitHub Pages, Vercel ou qualquer hospedagem estática

## Estrutura
```
kelly-landing/
  index.html
  kelly.apk          ← coloque o APK aqui
  screenshots/
    logo-k.png       ← logo do app
    1.png            ← Dashboard
    2.png            ← Lançamentos
    3.png            ← Assistente Kelly
    4.png            ← Contas
    5.png            ← Metas
    6.png            ← Investimentos
```
